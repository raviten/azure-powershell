# New-AzDataBoxEdgeBandwidthSchedule

## SYNOPSIS

## SYNTAX

```

 -StartTime <String> -StopTime <String> -Days <String[]> [-Bandwidth <Int32>] [-UnlimitedBandwidth]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```
